# 📊 Sales Overview Dashboard | Power BI

## 📌 Project Overview

This project presents an **interactive Sales Overview Dashboard** built using **Power BI**.
The dashboard analyzes business performance across four regions: **Central, East, South, and West**.

It allows users to dynamically explore **Sales, Profit, and Quantity** metrics for different years and compare **Current Year (CY)** performance with **Previous Year (PY)** performance.

The goal of this dashboard is to help businesses **monitor regional performance, identify sales trends, and make data-driven decisions.**

# 🎯 Business Requirements

The dashboard was developed to fulfill the following business needs:

### Regional KPI Monitoring

For each region (**Central, East, South, West**) the dashboard should:

* Display **Sales, Profit, and Quantity**
* Allow **dynamic metric selection**
* Show **Previous Year (PY) comparison**
* Provide **monthly sparkline trend visualization**
* Display **average line for trend analysis**

### Sales Distribution Analysis

* Visualize **sales distribution across states**
* Provide **state level comparison**
* Help identify **top performing states**

### Year over Year Performance Analysis

Provide a detailed table comparing:

* Current Year Sales
* Previous Year Sales
* YoY Sales Growth
* Current Year Profit
* Previous Year Profit
* YoY Profit Growth
* Current Year Quantity
* Previous Year Quantity
* YoY Quantity Growth

## 📂 Dataset Description

The dataset used in this project contains sales transaction data including customer details, product information, and financial metrics.

| Column Name | Description |
|-------------|-------------|
| Row ID | Unique identifier for each record in the dataset |
| Order ID | Unique identifier for each customer order |
| Order Date | The date when the order was placed |
| Ship Date | The date when the order was shipped |
| Ship Mode | Shipping method used for the order |
| Customer ID | Unique identifier assigned to each customer |
| Customer Name | Name of the customer who placed the order |
| Segment | Customer segment category (Consumer, Corporate, Home Office) |
| Country/Region | Country where the order was placed |
| City | City of the customer |
| State/Province | State or province of the customer |
| Region | Sales region (Central, East, South, West) |
| Product ID | Unique identifier for each product |
| Category | Main product category |
| Sub-Category | Product sub-category |
| Product Name | Name of the product |
| Sales | Total revenue generated from the sale |
| Quantity | Number of units sold |
| Discount | Discount applied to the product |
| Profit | Profit earned from the sale |

# 📊 Dashboard Features

## 1️⃣ Dynamic KPI Selection

Users can dynamically switch between:

* **Sales**
* **Quantity**
* **Profit**

using interactive buttons.

---

## 2️⃣ Year Filter

The dashboard includes a **Year Slicer** allowing users to analyze performance for:

* 2021
* 2022
* 2023
* 2024

---

## 3️⃣ Regional Performance Cards

Each region shows:

* Current metric value
* Previous year comparison
* Monthly performance trend
* Average monthly reference line

Regions included:

* Central
* East
* South
* West

---

## 4️⃣ Sales by State Map

A **bubble map visualization** shows:

* Sales distribution across states
* Larger bubbles represent higher sales

This helps quickly identify **high performing states**.

---

## 5️⃣ State Level Comparison

A **bar chart** is used to:

* Compare sales between states
* Identify top performing states
* Identify low performing states

---

## 6️⃣ Year-over-Year (YoY) Performance Table

A detailed comparison table includes:

| Metric     | Description            |
| ---------- | ---------------------- |
| CY Sales   | Current Year Sales     |
| PY Sales   | Previous Year Sales    |
| YoY Sales  | Sales Growth           |
| CY Profit  | Current Year Profit    |
| PY Profit  | Previous Year Profit   |
| YoY Profit | Profit Growth          |
| CY Qty     | Current Year Quantity  |
| PY Qty     | Previous Year Quantity |
| YoY Qty    | Quantity Growth        |

This table helps evaluate **business performance trends**.

Some insights that can be derived from the dashboard:

* South region has **significant profit growth**

* Monthly trends help identify **seasonal sales patterns**
